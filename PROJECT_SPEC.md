## 技術スタック

- **フレームワーク：** Astro（静的サイト・高速・多言語対応しやすい）
- **スタイリング：** Tailwind CSS
- **多言語：** Astro i18n（日本語 / 英語 / 繁体字中国語）
- **フォーム：** Formspree（メール受信用・無料）
- **画像最適化：** Astroの`<Image />`コンポーネントを使用

**方針：** モバイルファースト。明奈さんがコードを触らなくても更新できるよう、
テキストコンテンツはできる限り設定ファイル（JSONまたはMarkdown）に分離する。

## Claude Codeへの作業指示

以下の順番で作業してください：

1. `npm create astro@latest` でプロジェクト初期化
2. Tailwind CSS・Astro i18nを設定
3. 日本語・英語・繁体字の言語ファイル（JSONまたはYAML）を作成
4. 各ページをコンポーネント分割で実装
5. Formspreeのフォームを実装（送信先：asiance7hr@gmail.com）
6. `npm run build` でビルド確認
7. Vercel CLIでデプロイ（`vercel deploy`）
8. デプロイ後、お名前.comのDNS設定手順を日本語で出力する
